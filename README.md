#Veganologo
Um site de receitas veganas para pessoas que buscam ser mais saudaveis. A proposta do site é mostrar formas mais saudaveis e acessíveis pa noixxxxx. 

##Funcionalidades

* Cadastrar usuário
* Cadastrar receitas
* Apagar receitas
* Edita receitas
* Comentarios
* Carregar fotos
* Favoritos (talvez)

##Tecnologias

* HTML5
* CSS
* PHP
* SQL
